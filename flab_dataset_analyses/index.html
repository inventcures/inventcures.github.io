Zero-Shot PLM Performance (FLAB-002)

Can ESM-2 embeddings predict antibody fitness without fine-tuning? Short answer: No.

ESM-2 Correlation with Experimental Properties
Pearson correlation (r) and Spearman rank correlation (ρ) for n=300 sequences per property.
Property Pearson r Spearman ρ AUROC Interpretation
Thermostability -0.140 -0.143 0.42 Worse than random
Expression -0.060 -0.267 0.35 Worse than random
Binding Affinity 0.010 0.175 0.56 Weak signal
Aggregation -0.052 -0.088 0.46 No predictive value
Key finding: Zero-shot ESM-2 embeddings show negative correlations with most fitness properties. This confirms the FLAb paper's finding that task-specific fine-tuning is essential for antibody property prediction.

Method: Mean-pooled ESM-2 (650M) embeddings with linear regression. Data: FLAb benchmark experimental measurements.

Critical insight: Humanness alone (AUROC 0.284) is worse than random chance at predicting immunogenicity. This challenges the common assumption that "more human = less immunogenic." Multi-feature models using sequence properties achieve meaningful prediction (AUROC 0.737).

Methods

How we analyzed the FLAb benchmark data.

Data Source

FLAb benchmark from Gray Lab (Johns Hopkins). 160 CSV files covering thermostability, expression, binding, and aggregation.

Embeddings

ESM-2 (650M parameters) with mean pooling over sequence length. GPU inference on Modal cloud.

Developability

TAP2 tool for liability motif detection, hydrophobicity, charge, and aromatic content analysis.

Statistical Tests

Welch's t-test with Bonferroni correction. Cohen's d for effect sizes. Bootstrap CI for correlations.
